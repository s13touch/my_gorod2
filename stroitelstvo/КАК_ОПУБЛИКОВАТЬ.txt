Бесплатный хостинг — как получить ссылку для заказчика
========================================================

Вариант 1: Netlify (проще всего, ссылка за 1 минуту)
-----------------------------------------------------
1. Зайдите на https://app.netlify.com
2. Зарегистрируйтесь (бесплатно): через email или GitHub/Google
3. Перейдите на https://app.netlify.com/drop
4. Перетащите в окно браузера папку «stroitelstvo» (целиком)
   Или перетащите все файлы из неё: index.html, styles.css, script.js
5. Netlify выдаст ссылку вида: https://случайное-имя.netlify.app
6. Эту ссылку можно отправить заказчику


Вариант 2: GitHub Pages (если репозиторий уже на GitHub)
--------------------------------------------------------
1. Залейте проект на GitHub (если ещё не залит)
2. В репозитории: Settings → Pages
3. Source: Deploy from a branch
4. Branch: main, папка: / (root)
5. В корень репозитория поместите файлы из папки stroitelstvo
   (index.html, styles.css, script.js), чтобы главная страница открывалась по ссылке
6. Ссылка будет: https://ваш-логин.github.io/my_gorod2/


Вариант 3: Cloudflare Pages
----------------------------
1. Зайдите на https://pages.cloudflare.com
2. Войдите или зарегистрируйтесь
3. Create a project → Direct Upload
4. Перетащите папку stroitelstvo или загрузите zip с её содержимым
5. Деплой создаст ссылку вида: https://название.pages.dev
